# Задание 1. Описание тематики дипломного проекта

**Тема дипломного проекта:**  
«Разработка веб-приложения интернет-магазина электроники с функционалом поиска и регистрации пользователей»

---

## 1. Актуальность темы

В современных условиях цифровой экономики интернет-торговля занимает значительную долю потребительского рынка. Покупка электроники онлайн стала стандартной практикой для миллионов пользователей. Однако многие малые и средние интернет-магазины не имеют собственных программных решений и вынуждены использовать платформы с ограниченными возможностями или высокими комиссиями.

Разработка собственного веб-приложения интернет-магазина позволяет:

- предложить пользователю удобный и интуитивно понятный интерфейс;
- реализовать быстрый поиск товаров по ключевым характеристикам;
- обеспечить персонализацию через систему регистрации и авторизации;
- заложить основу для дальнейшего масштабирования проекта.

Таким образом, создание интернет-магазина электроники с базовым функционалом поиска и регистрации является **актуальной задачей**, имеющей как учебную, так и практическую ценность.

---

## 2. Целевая аудитория

Целевая аудитория разрабатываемого веб-приложения включает две основные категории пользователей:

| Категория | Описание |
|-----------|----------|
| **Зарегистрированные покупатели** | Пользователи, создавшие учётную запись. Получают доступ к персонализированным функциям: сохранение корзины, история заказов, ускоренное оформление покупок. |
| **Гости (неавторизованные пользователи)** | Пользователи, не прошедшие регистрацию. Имеют доступ к просмотру каталога, поиску товаров и добавлению товаров в корзину, но без сохранения данных между сеансами. 
---
## 3. Предполагаемые функциональные возможности (MVP)
В рамках минимально жизнеспособного продукта (MVP) предусмотрена реализация следующего функционала:

### 3.1. Каталог товаров
- Отображение списка товаров (название, цена, изображение, краткое описание).
- Возможность просмотра детальной карточки товара.

### 3.2. Поиск и фильтрация
- Поиск товаров по названию в реальном времени (без перезагрузки страницы).
- Фильтрация товаров по категориям (смартфоны, ноутбуки, аксессуары).

### 3.3. Корзина
- Добавление товаров в корзину с выбором количества.
- Удаление товаров из корзины.
- Автоматический подсчёт итоговой суммы.

### 3.4. Регистрация и авторизация
- Форма регистрации (имя, email, пароль).
- Форма входа в существующую учётную запись.
- Валидация полей на стороне клиента.
- Хранение данных пользователя в локальном хранилище (localStorage).

### 3.5. Интерфейс
- Адаптивная вёрстка, корректно отображающаяся на устройствах с различным разрешением экрана.
- Интуитивно понятная навигация между страницами.

---

## 4. Используемые технологии

В рамках дипломного проекта используются следующие технологии и инструменты:

| Компонент | Технология | Назначение |
|-----------|------------|------------|
| **Вёрстка** | HTML5, CSS3 | Структура и стилизация страниц |
| **Клиентская логика** | JavaScript (ванильный) | Поиск, фильтрация, работа с корзиной, регистрация |
| **Хранение данных** | localStorage | Хранение данных пользователя и корзины |
| **Визуализация** | Flexbox / Grid | Построение адаптивных макетов |
| **Контроль версий** | Git + GitHub | Хранение исходного кода и отчётности |

По завершении проектирования и разработки будут получены следующие результаты:

1. **Рабочее веб-приложение** интернет-магазина электроники, включающее:
   - главную страницу с каталогом товаров;
   - страницу корзины с возможностью управления заказом;
