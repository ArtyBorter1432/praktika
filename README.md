## 1. Описание проекта

В рамках дипломного проекта разработано веб-приложение интернет-магазина электроники, реализующее базовый функционал электронной коммерции.

### 1.1. Целевая аудитория

| Категория | Описание |
|-----------|----------|
| Зарегистрированные покупатели | Пользователи, имеющие возможность оформления заказов и сохранения корзины |
| Гости | Пользователи, осуществляющие просмотр каталога и поиск товаров без авторизации |

### 1.2. Основной функционал

- Просмотр каталога товаров с пагинацией
- Поиск товаров по названию
- Фильтрация товаров по категориям
- Добавление товаров в корзину
- Управление корзиной (изменение количества, удаление)
- Регистрация и авторизация пользователей
- Персонализированные данные (сохранение корзины, профиль)

## 3. Инструкция по запуску

### 3.1. Системные требования

- Операционная система: Windows / Linux / macOS
- Браузер: Google Chrome, Mozilla Firefox, Opera, Яндекс.Браузер (последние версии)
- Интернет-соединение: не требуется (все данные хранятся локально)

### 3.2. Порядок запуска

1. Скопировать репозиторий или скачать архив с проектом
2. Перейти в директорию проекта
3. Открыть файл `index.html` в любом веб-браузере
